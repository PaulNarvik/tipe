Paramètres

nLe pointeur vers le premier grand entier à sommer.

Précondition

Note

◆ largeint_resize()

void largeint_resize ( largeint * n,
size_t new_size )

Redimensionne le grand entier à la taille donné.

Paramètres
nLe pointeur vers le grand entier à redimensionner.
new_sizeLa nouvelle taille de l'entier.
Précondition
La nouvelle taille est supérieure au nombre de chiffres utilisés, pour éviter toute perte d'informations.
Avertissement
Dans le cas où la précondition n'est pas respectée, les chiffres restants seront tronqués.
